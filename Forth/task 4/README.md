## Deploy on Vercel

1. Go to [Vercel](https://vercel.com)  
2. Log in and click **"New Project"**  
3. Import your GitHub repository  
4. **Framework Preset:** Select **Other**  
5. **Build Command:** Leave empty  
6. Click **Deploy**
